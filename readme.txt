This is an IoT-ML Based Project for Crop Predication based on Soil and Weather Condition.

It Predict Crop for Particular Location Based on Previous Crop Histroy of same location and Soil Testing Report.

Technology Stack:
    1) Android
    2) PHP-ML Library
    3) Bolt-IoT Hardware Kit
    4) MongoDB
